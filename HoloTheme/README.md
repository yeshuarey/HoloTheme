# 🐺 Holo's Wisdom - Spice and Wolf Theme

A warm, rustic, and elegant dark theme for Visual Studio Code, inspired by the colors of **Holo the Wise Wolf**. 

This theme brings the atmosphere of the merchant paths and the golden wheat fields of Pasloe directly to your editor.

## 🎨 Features
- **Harvest Gold**: Highlights for important variables and constants.
- **Wolf's Fur Red**: Warm tones for keywords and operators.
- **Rustic Wood**: A deep, comfortable dark background for long coding sessions.

## 🚀 Installation
1. Open **Visual Studio Code**.
2. Go to the **Extensions** view (`Ctrl+Shift+X`).
3. Search for `Holo's Wisdom` or install the `.vsix` manually.
4. Enjoy the wisdom of the wolf!
